to run the program, open the folder in terminal and run the command: "node backend.js",
then go to your web browser and type in "localhost:3003" and click enter.
In order to play a game, you must have 3 players, which can be achieved by having 3 separate windows each from a different google account on the website.
Make sure each player gets their own login, and joins the same room, when ready click start.
